# Compteur de mots

Projet Java simple permettant de compter le nombre de mots dans un fichier texte.

## Fonctionnalités

- Ouvrir un fichier texte
- Lire le fichier ligne par ligne
- Compter le nombre total de mots
- Afficher le résultat dans la console

## Langage utilisé

- Java

## Structure du projet

- Main.java
- test.txt

## Exécution

1. Ouvrir le projet dans IntelliJ IDEA
2. Ajouter un fichier `test.txt` à la racine du projet
3. Lancer la classe Main
4. Le programme affiche le nombre total de mots du fichier
